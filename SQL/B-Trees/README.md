# 🌳 B-Tree & B+ Tree

## 📖 Overview
B-Tree and B+ Tree are self-balancing tree data structures used for efficient searching, insertion, and deletion of data. They are widely used in database indexing and file systems.

## 🔍 Difference Between B-Tree and B+ Tree
| B-Tree | B+ Tree |
|--------|----------|
| Data stored in all nodes | Data stored only in leaf nodes |
| Faster for single record search | Better for range queries |
| Leaf nodes are not linked | Leaf nodes are linked |
