# Data Cleaning, Preparation and Merging

## Overview
Data cleaning is the process of improving data quality by handling missing values, duplicates, incorrect formats, and inconsistent data. Data preparation organizes data for analysis, while merging combines multiple datasets.

## Topics Covered
- Data Cleaning
- Data Preparation
- Data Merging

## Data Cleaning
Performed operations such as:
- Handling missing values
- Removing duplicate records
- Correcting inconsistent values
- Standardizing formats

## Data Preparation
Prepared datasets by:
- Selecting required columns
- Renaming columns
- Changing data types
- Formatting data for analysis

## Data Merging
Learned how to combine datasets using different merge operations based on common columns.

## Learning Outcome
Learned how to prepare clean and structured datasets before performing data analysis.
