# Pandas
## Overview
This folder contains my hands-on practice while learning **Pandas** for data analysis. The examples cover data cleaning, preprocessing, manipulation, feature engineering, joins, and exploratory data analysis.

## Topics Covered
### Data Cleaning
- Detect missing values
- Count missing values
- Drop rows with missing values
- Drop columns with missing values
- Drop rows based on specific columns
- Fill missing values using:
  - Mean
  - Median
  - Forward Fill (ffill)
  - Backward Fill (bfill)
### Data Preparation
- Convert text to lowercase
- Convert text to uppercase
- Replace strings
- Replace values
- Regular expression replacement
### Data Type Conversion
- Integer conversion
- Float conversion
- String conversion
- Datetime conversion
- Numeric conversion
- Error handling using `errors='coerce'`
### Feature Engineering
- Z-Score calculation
- IQR calculation
- Outlier detection using Z-Score
- Outlier detection using IQR
### Data Merging
- Inner Join
- Left Join
- Right Join
- Outer Join
### Data Analysis
- Univariate Analysis
- Bivariate Analysis
