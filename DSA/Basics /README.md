# DSA Basics

### Topics Covered
- What is an Algorithm?
- Time Complexity
- Space Complexity
- Big O Notation

### Time Complexities Learned
- O(1)
- O(n)
- O(n²)

### Key Takeaways
- Time complexity measures how an algorithm grows with input size.
- Constants are ignored in Big O notation.
- Nested loops are not always O(n²); it depends on how many times each loop runs.
